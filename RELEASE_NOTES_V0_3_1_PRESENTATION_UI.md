# v0.3.1 Presentation UI

`v0.3.1-presentation-ui` is a presentation and reviewability release for Financial Agent Evidence OS.

It does not change the v0.3 evidence, strategy result, real-market gate, or claim boundary. The `v0.3-real-market-data-defense` release remains the sealed real-market evidence milestone.

## Highlights

- Adds an executive dashboard summary for the read-only Streamlit evidence viewer.
- Surfaces real-market evidence status, sealed-data mode, ETF coverage, benchmark period, and data distribution policy on the first dashboard screen.
- Adds a three-minute demo script for presentation use.
- Adds a presentation screenshot for README and release pages.
- Keeps weak candidate real-market performance visible and preserved as evidence.

## Claim Boundary

This release does not claim live trading readiness, future-return prediction, return guarantees, broker routing, realized investor returns, or market dominance.

The product claim remains:

> Financial Agent Evidence OS verifies, scopes, and seals financial AI performance claims using deterministic and sealed evidence packets.

## Relationship to v0.3

`v0.3-real-market-data-defense` proves that the Evidence OS can run on sealed real ETF data. `v0.3.1-presentation-ui` makes that result easier to review in a dashboard, README, and short demo.
