# Performance Non-Claims

The performance layer explicitly does not claim:

- Financial advice.
- Live trading readiness.
- Return guarantees or future performance.
- Realized investor returns.
- Broker integration, order routing, or execution readiness.
- Universal market, strategy, or external-framework dominance.
- Any performance result outside the included `downside_performance_v1` benchmark suite.

All public wording must preserve these boundaries.
